Commands

Here is information on the commands that are aviable for the discord bot.
At the moment, "/" commands are not aviable
Prefix is "!"
Currently this bot dose not support "/" commands
Spaltting
To attempt to splat the salmon you must say !splat, followed by the salmons name (Shown in embed)

!splat [salmon_name]
Shop
When you say shop it will show all items listed shown as so: "[#] [Name] X[Quantity] ([CMD]) | [Cost]"

!shop
Buy
To buy an item you must specificy the number listed in the shop

!buy [#]
Events
This command will allow you to see the current or future events

!event
