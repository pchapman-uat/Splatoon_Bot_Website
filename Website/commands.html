Commands

Here is information on the commands that are available for the discord bot.
At the moment, "/" commands are not available

Splatting
To attempt to splat the salmon you must say !splat, followed by the salmons name (Shown in embed)

!splat [salmon_name]
Shop
When you say shop it will show all items listed shown as so: "[#] [Name] X[Quantity] ([CMD]) | [Cost]"

!shop
Buy
To buy an item you must specify the number listed in the shop

!buy [#]
Inventory
This command will show your current Inventory, listed as [Name] | [CMD] | [#]

!inv
Events
This command will allow you to see the current or future events

!event
Items (Combat)
To use a combat related item, such as a Booyah Bomb, you must say it while splatting an item. Note: This may change in the future.

!splat [salmon_name] item [CMD]
Items (Other)
For other items, you must use the item command. Note: This may change in the future.

!item [CMD]
Leaderboard
This command will show the current Leaderboard. Note: This soon will be added to this website!

!leaderboard
Event Leaderboard
This command will show the current event Leaderboard, this is different as it will show the temporary event leaderboard. Note: This soon will be added to this website!

!leaderboard
Stats
View your stats and compare against friends!

!stats
